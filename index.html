Dice Puzzle Page

Dice Puzzle is a game released in 1983 for the Atari 2600. It is a quite simple puzzle and is somewhat trivial to solve. It never became very popular and because of this there is very little information about it on the internet. The goal of this web page is to list as much information as possible about the puzzle, and be an interesting source of information.

The rules of the game are quite simple. It consists of a 4x4 grid filled with dice. Each row, column and diagonal can be incremented by 1, rolling over to 1 if 6 is reached. The goal, according to the instruction manual, is to change all of the dice to 6. An example of a possible position is shown below.
 ⎯⎯⎯⎯⎯⎯⎯
|5 4 4 4
|2 5 3 3
|1 2 4 2
|5 6 6 2

There are 10 possible moves that can be made, however because they simply increment a certain portion of the board by 1, there is no movement of pieces, and the order of the moves does not matter. To describe sequences of moves, an array of lenth 10 will be used, with index 0 corresponding to the top-right diagonal, and moving counter-clockwise down the puzzle. Also, every dice, or piece in the puzzle will be labelled a through p, as shown below:

5 4⎯3⎯2⎯1⎯0
6|a b c d
7|e f g h
8|i j k l
9|m n o p

0 and 5 correspond to the diagonals, 1-4 the columns, and 6-9 the rows. The sequence of moves executed on the puzzle at this point would be reperesented as [A,B,C,D,E,F,G,H,I,J], mod 6, as executing a given move 6 times is equivalent to zero. The state of the puzzle itself can also be written mod 6, so long as 1 is subtracted from the value of each piece. The solution to the first puzzle listed can be written as [3 3 0 0 4 1 2 0 1 0].

Solution:

There are a couple of different ways the puzzle can be solved, however the one that I view as most efficient is listed below:

z = (c+g+h-d)/2
solution = [z-g, c+g-(z+d), 0, c-b, c+g+h-(z+d+e), z+b - (f+c), 6-c, 6-z, 6-(i+z-e), 6-o]

I will explain how this works later, but having the solution in such a linear way makes the solve much more straigtforward, making it faster to solve. I will elaborate on this later, however that is a very systematic way of doing things.

Although according to the instruction manual, the goal is to set the puzzle to a state of all sixes, when playing the game this is incorrect. As long as all of the dice match, it will be considered solved regardless if they are all 6 or not. This will be mentioned further.


Quiet Patterns:

Like Lights Out, Dice Puzzle has several quiet patterns. Quiet patterns are sequences of moves which have no effect on the board. There are a total of 11 quiet patterns on the Dice Puzzle (12 if doing nothing is included), which are as follows:
[0111105555] [3411435225]
[0222204444] [3522534114]
[0333303333] [3033033003]
[0444402222] [3144132552]
[0555501111] [5150105023]
[3300330330]

To find an optimal solution is trivial using quiet patterns. All that is needed to find an optimal solution is to find any solution, then apply each quiet pattern to the sequence to see if the total number of moves is reduced. The quiet patterns provide every possible sequence of moves to reach a given state.
This also allows for a simple calculation of the number of possible positions on the puzzle. There are 6^10 possible sequences of moves, dividing that by 12 quiet patterns gives 5038848 positions in the puzzle.

When solving the puzzle relatively, as in actually playing the game, it is useful to consider positions that do not affect the relative state of the board. There are a total of 72 'relative quiet patterns' (RQPs) in Dice Puzzle (6 x 12). To find an optimal solution relatively, do the same as you would for absolute, just with the greater number of quiet patterns. I will list here all of the relative quiet patterns soon.

Extensions:

There are many ways the dice puzzle can be extended, some ideas are listed below:
- As mentioned before, using bases other than 6 is an interesting way to extend the puzzle. Essentially the same methods as base 6 can be used on any other base, however it should be noted that odd bases will have half as many quiet patterns, and therefore twice as many positions relative to the base. I have no proof for this, but I believe even bases above 6 also have only 2b quiet patterns, and odd bases b quiet patterns. The number of RQPs is simply that multiplied by the base again. I may be wrong, please feel free to verify. I will soon add an entire section dedicated to other bases, when I find more time.
- Different board sizes could be used. Because of the diagonals it might only make sense to use squares, however it could possibly be altered to accomodate that. I have not explored this possibility at all, however the only major difference I can forsee is the middle square on odd bases being interesting. I will definitely explore the 3x3, and maybe the 5x5 board soon.
- Perhaps adding an increment to puzzles like Cary Huang's loopover would be interesting. I am not sure what affect this would have, however it would make it more analogous to a Rubik's cube, with the increment acting as orientations. I have no idea how this would actually work, it is just an interesting idea.
- There are an infinite number of similar puzzles that could be created by simply using an array of some length, and an arbitrary number of operations that increment certain positions in the array by a certain amount, mod some number. Lights Out would be considered in this very loose definition. I have no idea what the implications of this would be, but it is an interesting thought.

 


Instruction Manual Text:


DICE PUZZLE

The dice puzzle has 16 squares of random numbers between 1-6. The object is to change all the numbers to 6 in the shortest amount of time, using the fewest numbers of turns.

When the power is turned on, each square will have the number 1. After pushing the reset button, all the dice change to random numbers. (The game is guaranteed to be solvable every time.) There is a small square that moves outside of the left side and over the top by using the joystick. When the fire button is pushed, all the numbers in the row next to the square increase by one. The numbers can also be changed diagonally. Every 5 minutes a warning will sound, up to a maximum time limit of 60 minutes. When all the dice are changed to 6, the game will automatically end displaying time used and number of turns taken.

The dice puzzle - sometimes mind boggling, sometimes a cinch, is always loads of fun. Come and match wits with the computer.
